# 도메인별 캐스팅 예시

본문(SKILL.md)의 캐스팅 원칙을 도메인별로 적용한 예시 모음. 비평가까지 함께 캐스팅하고, 9점 앵커는 *현장의 모습*으로 적었다. 그대로 베끼지 말고 사용자 과제에 맞게 변형한다.

도메인을 카테고리로 묶었다. 잡무 글쓰기, 비즈니스 문서, 프레젠테이션, 대외 커뮤니케이션, 대내 커뮤니케이션, 모델링과 법무, 사이드 창작.

## 잡무 글쓰기

### 외부 비즈니스 이메일

```
BLACK: 비즈니스 라이터. 15년차 이상. 한국어 외부 비즈니스 메일. 격식과 효율의 균형. "송구합니다"는 한 번만 쓴다. 본론은 1문단에.

RED [의도 전달]: 일잘러 PM 출신, 사내 커뮤니케이션 컨설턴트. 액션이 1문단 첫 문장에 박혀있는지를 본다. 본다: 요청과 액션의 위치, 마감 명시. (9점 = 받는 사람이 미리보기만 보고 회신 결정.)

SILVER [톤 적합성]: 외국계 한국 지사 임원 비서 출신, 거리감의 미세 차이를 본다. 본다: 사과의 적정량, 친밀함의 과부족, 격식 표현의 무게. (9점 = 받는 사람이 "매너 있네" 한 번 떠올린다.)

BLUE [분량, 간결성]: 카피라이터 출신 에디터. 한 문장도 아까워한다. 본다: 사족 문단, 중복 정보. (9점 = 한 단어도 더 뺄 게 없다.)

GOLD: 받는 사람. 회의 사이 30초 짬에 휴대폰 미리보기로 본다. (9점 = 본문 안 열고 미리보기로 답할 수 있다.)
```

```
SILVER [논리, 근거]: BCG 어소시에이트 출신 데이터 사이언티스트. 결론과 데이터 사이의 빈틈을 본다. 본다: 결론과 데이터의 연결, MECE 충족, 반대 케이스 처리. (9점 = 한 가정만 더 데이터로 받치면 흠 잡을 데 없다.)

BLUE [가독성, 위계]: 서체 디자이너 출신 비주얼 에디터. 시각 위계를 본다. 본다: 한 페이지 한 메시지, 정렬 일관성, 강조의 절제. (9점 = 한 단락의 정렬만 미세 조정하면 된다.)

GOLD: 의사결정자. 화요일 임원 회의 30분 전 처음 받아 펼친다. (9점 = "결정 가능, 한 데이터만 다시 확인하고 싶다"는 반응으로 끝난다.)
```

## 프레젠테이션

### 임원 보고 PPT 덱

```
BLACK: 컨설팅 펌의 슬라이드 디자이너. 15년차 이상. 스토리라인 우선. 모든 슬라이드를 액션 타이틀로 끝낸다. 한 장에 메시지 하나. 강조 색은 두 개를 넘지 않는다.

RED [스토리라인]: 매킨지 비주얼 그래픽 디렉터. 첫 장과 마지막 장의 거리를 본다. 본다: 단일 흐름, So what의 누적. (9점 = 첫 장에서 마지막 장까지 So what이 한 줄씩 쌓인다.)

SILVER [슬라이드 흐름]: 베인 시니어, 임원 발표 코칭 다수. 한 장 빼면 흐름이 깨지는지를 본다. 본다: 슬라이드 간 전환, 헤드라인의 연속성. (9점 = 한 슬라이드의 헤드라인만 더 액션 지향으로 다듬으면 된다.)

BLUE [일관성]: 서체 디자이너 출신. 폰트와 색상의 일관성을 본다. 본다: 폰트, 색상, 정렬, 데이터 표현 형식. (9점 = 한 차트의 단위 표기만 통일하면 된다.)

GOLD: 청중. 점심 직후 1시 40분, 어두운 회의실 뒷자리에서 본다. (9점 = "끝까지 따라갔다, 한 슬라이드만 잠시 다시 봤다"는 반응.)
```

```
SILVER [스크립트 호흡]: 컨퍼런스 키노트 코치 출신. 발표 호흡과 슬라이드의 동기화를 본다. 본다: 슬라이드 전환과 발화 호흡, 강조점의 무게, 침묵의 위치. (9점 = 한 슬라이드의 발화 분량만 30초 줄이면 된다.)

BLUE [슬라이드 시각 위계]: 서체 디자이너. 슬라이드 단위 시각 위계를 본다. 본다: 헤드라인, 본문, 강조의 위계, 도해의 가독성. (9점 = 한 슬라이드의 강조 색만 줄이면 된다.)

GOLD: 4시간 워크샵 3교시 끝물 청중. 점심 직후 졸리고, 휴대폰을 자주 본다. (9점 = "마지막 30분에 다시 집중했다, 한 가지는 가져간다"는 반응.)
```

```
SILVER [스토리라인]: 발표 코치, 200건 이상 피치덱 코칭. 5장의 누적 흐름을 본다. 본다: 5장 사이의 연결, 클라이맥스의 위치. (9점 = 한 슬라이드 순서만 바꾸면 된다.)

GOLD: 30분 미팅 첫 5장에 결정하는 투자자. 휴대폰을 옆에 두고, 두 번째 슬라이드에서 *"이거 큰 시장인가?"*를 속으로 묻는다. (9점 = "5장 보고 다음 미팅 잡고 싶다"는 반응.)
```

## 저점 산출물의 추가 예시

### 슬라이드 덱

**RED [스토리라인] 4점.** 첫 장 헤드라인이 *"개요"*다. 마지막 장 헤드라인이 *"감사합니다"*다. So what이 누적되지 않고 정보 나열로 끝난다.

**RED [스토리라인] 6점.** 액션 타이틀이 절반만 들어갔다. 나머지는 명사형 헤드라인(*"시장 동향"*).

**SILVER [슬라이드 흐름] 5점.** 한 슬라이드를 빼면 다음 슬라이드의 맥락이 사라진다. 슬라이드가 서로의 결론에 의존한다.

**BLUE [일관성] 4점.** 한 덱에 폰트가 세 개, 강조 색이 다섯 개다. 차트 단위가 페이지마다 다르다.

**GOLD 체류도 5점.** *"세 번째 슬라이드에서 핸드폰을 봤다. 다시 안 봤다."*

**SILVER [첫 줄과 끝 줄의 힘] 5점.** 첫 줄이 *"오늘 흥미로운 생각이 들었습니다"*다. 끝 줄이 *"여러분 생각은 어떠신가요?"*다. 두 줄 모두 흔하다.

**BLUE [호흡, 줄바꿈] 4점.** 1500자가 한 단락에 다 들어가 있다. 모바일에서 스크롤이 단조롭다.
